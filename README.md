# Wind Turbine Engineering Project Website

This repository contains the website for our wind turbine engineering design project.  
The website summarizes our turbine design process, experimental results, and electrical
power measurements collected during the project.

## Project Overview

The goal of this project was to design and test a small-scale wind turbine and analyze
how design factors affect electrical power generation.

Throughout the project we tested:

- Blade number
- Blade angle
- Blade material
- Electrical load (resistor values)

Voltage generated by the turbine was measured using an Arduino and analyzed using Matlab
to determine the power output of the system.

## Website Contents

The website includes:

- Project summary
- Team member information
- Photos of the turbine design
- Arduino measurement setup
- Matlab voltage graph
- Explanation of the power measurement process
- Table of power output for different resistors
- Links to design journals documenting each stage of development
